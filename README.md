# 🍽️ foodZeep

**Reducing food wastage by connecting surplus food from restaurants, hotels, and events with buyers at discounted prices.**

---

## 📖 About

foodZeep is a full-stack application that helps:
- **Sellers** (Restaurants, Hotels, Event organizers) list surplus food
- **Buyers** purchase quality food at discounted prices
- **Reduce food wastage** while helping communities

---

## 🛠️ Tech Stack

**Backend:**
- Node.js
- Express.js
- MySQL
- JWT (Authentication)
- REST API

**Tools:**
- Postman (API Testing)
- Git (Version Control)
- VSCode (Editor)

---

## 👥 User Roles

- **Admin** - Manage platform, oversee operations
- **Seller** - Create listings, manage inventory, accept orders
- **Buyer** - Browse items, place orders, make payments

---

## ✨ Features

### Authentication
- User registration and login
- Role-based access control (Admin/Seller/Buyer)
- JWT-based authentication

### For Sellers
- Create and manage restaurant/hotel profiles
- Add food items with details (ingredients, quantity, expiry, discount)
- View and accept orders
- Payment history tracking

### For Buyers
- Browse available food items
- View item details (ingredients, expiry, discount)
- Place orders
- Make payments
- Order history

### For Admin
- Manage users and sellers
- Platform oversight
